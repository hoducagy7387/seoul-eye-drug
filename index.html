Google Sheets 연동 설정
아래 순서대로 설정하면 접수실·수술실·연구실 어느 PC에서 열어도
모두 같은 데이터가 실시간으로 공유됩니다.
1
Google 스프레드시트 만들기
sheets.new 에 접속하여 새 스프레드시트를 만드세요.
이름은 자유롭게 지정 (예: 서울안과 약품재고).
2
Apps Script 편집기 열기
스프레드시트 상단 메뉴에서
확장 프로그램 → Apps Script 클릭
3
아래 코드를 복사해서 붙여넣기
기존 코드를 모두 지우고 아래 코드로 교체하세요.
// ★ 이 코드를 Apps Script에 붙여넣으세요 ★ function doGet(e) { var action = e.parameter.action || 'load'; var ss = SpreadsheetApp.getActiveSpreadsheet(); var sheet = getOrCreate(ss, 'DATA'); var result; try { if (action === 'load') { result = { drugs : parse(sheet.getRange('A1').getValue()), log : parse(sheet.getRange('B1').getValue()), mgrs : parse(sheet.getRange('C1').getValue()) }; } else if (action === 'save') { var type = e.parameter.type; var data = e.parameter.data; if (type === 'drugs') sheet.getRange('A1').setValue(data); if (type === 'log') sheet.getRange('B1').setValue(data); if (type === 'mgrs') sheet.getRange('C1').setValue(data); result = { ok: true }; } else { result = { error: 'unknown action' }; } } catch(err) { result = { error: err.toString() }; } return ContentService .createTextOutput(JSON.stringify(result)) .setMimeType(ContentService.MimeType.JSON); } function parse(val) { try { return val ? JSON.parse(val) : null; } catch(e) { return null; } } function getOrCreate(ss, name) { return ss.getSheetByName(name) || ss.insertSheet(name); }
4
웹 앱으로 배포
Apps Script 편집기 우측 상단 배포 → 새 배포 클릭
① 유형: 웹 앱 선택
② 액세스 권한: "모든 사용자" 선택
③ 배포 버튼 클릭 → 권한 승인 → 완료
5
웹 앱 URL 붙여넣기
배포 완료 후 표시되는 URL을 아래에 입력하세요.
URL을 입력하고 연결 확인 버튼을 누르세요.
